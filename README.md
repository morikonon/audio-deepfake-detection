# 🎧 Audio Deepfake Detection: Classical vs Deep Learning

This project explores the task of **audio deepfake detection** by comparing two fundamentally different approaches:

- **Classical Machine Learning** using handcrafted features (MFCC)
- **Deep Learning models** trained on audio representations (e.g., spectrograms)

The goal is to build a **reproducible research pipeline** and analyze the strengths and weaknesses of each approach.
